Projects

ESA: Analytics

ESA: Analytics is a subscription based progressive web app used to visualize and analyze geological well data in the continental United States. The tech stack for this project includes React, Next, Plotly, Tanstack, Express, MongoDB, and AWS. While this app has not been released for public use, it has been used internally to increase the speed in which the geologist can quality control the large amounts of well data in out database.
